## Quran Search

Created this simple app for personal usage that searches for the given text in the Noble Quran and displays the details.

## Tech and Tools
- [X] Java
- [X] Retrofit
- [X] Android Studio
- [X] 3rd party API [AlQuran API](https://api.alquran.cloud/v1/)
